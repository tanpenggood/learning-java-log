# 学习Java日志体系

## 项目结构

```
├─java-log-impl           # Java各个日志实现（jul、log4j、log4j2、logback）
├─java-log-api-jcl        # commons logging
├─java-log-api-slf4j      # slf4j
└─log4j2-vulnerabilities  # log4j2安全漏洞复现
```
